* Expose high-level methods like `deploy()`, `retract()`, `setRollerSpeed(rps)`.
* Expose getters like `pivotAngleDegrees`, `rollerSpeedRps`, `isDeployed`.

#### [NEW] [AresRobot.kt](file:///c:/Users/david/dev/robotics/ftc/ARESLib-Kotlin/core/src/main/kotlin/com/areslib/subsystem/AresRobot.kt)
* Create `AresRobot(hardwareIO: HardwareIO)` container class.
* Expose instantiated facades: `drive`, `shooter`, `intake`.
* Expose an `.update()` method:
  1. Reads all physical hardware sensors.
  2. Dispatches update actions to Redux store.
  3. Commands hardware outputs based on updated state.
  4. Automatically records state telemetry to AdvantageScope/NT4.
