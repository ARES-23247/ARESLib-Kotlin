# Requirements - Milestone v2.8: Deterministic Input Replay & "What-If" Ghost Simulation

Active requirements for Milestone v2.8.

## Active Requirements

### Category: Input Replay Core
* [x] **IO-01 (Subsystem IO Boundaries)**: Platform-independent SubsystemIO interfaces and inputs structures for decoupled core validation.
* [x] **REC-01 (Thread-safe Raw Input Logger)**: Thread-safe raw input logger writing microsecond-accurate JSONL logs to disk.
* [x] **REP-01 (Offline Replay Runner)**: Offline replay runner executing input logs directly through EKF pose estimation.

### Category: Visualization & UI
* [x] **GUI-01 (Replay GUI Dashboard)**: Native Compose Multiplatform desktop application visualizing real vs. ghost robot tracks with interactive tuning sliders.
* [x] **TEL-01 (AdvantageScope Dual Pose Streaming)**: Simultaneously stream real and ghost robot poses via NT4 to AdvantageScope.
