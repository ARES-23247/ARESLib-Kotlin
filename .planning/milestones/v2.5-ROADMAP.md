# Milestone v2.5: Hardened EKF Localization & Dynamic Sensor Fusion (Archived Roadmap)

## Goals & Objectives
Hardening robot localization, EKF pose estimation, outlier filters, and dynamic covariance scaling under slippage, tilt, speed, and distance limits.

## Completed Phases

- [x] **Phase 49: State Expansion & IMU Pitch/Roll Telemetry** (Completed 2026-05-18)
  - Added spatial physical telemetry properties (pitch, roll, linear acceleration, angular velocity) to `DriveState`, `DriveHardwareUpdate`, and `PoseUpdate`.
  - Upgraded RootReducer to map telemetry fields dynamically.

- [x] **Phase 50: Advanced Outlier Filter (3D Boundaries, Angular Speed, Acceleration/Shock Lockouts)** (Completed 2026-05-18)
  - Built 3D spatial field boundaries checks (X/Y perimeter, Z height bounds).
  - Implemented high angular velocity motion blur rejections ($>2.0\text{ rad/s}$).
  - Configured linear shock collision lockouts ($>2.5\text{ G}$) to protect EKF against sudden impacts.

- [x] **Phase 51: Dynamic Odometry Covariance Scaling (Tilt/Beaching Protection)** (Completed 2026-05-18)
  - Monitored pitch and roll to scale up EKF process noise covariance ($Q$) by $100\times$ when absolute chassis tilt exceeds $8^{\circ}$.
  - Froze encoder odometry updates when tilt exceeds $15^{\circ}$ (beached chassis).

- [x] **Phase 52: Dynamic Vision Covariance Scaling (Distance-based SD & Multi-Tag Trust)** (Completed 2026-05-18)
  - Applied quadratic standard deviation distance growth ($\sqrt{1.0 + d^2}$) to trust remote observations less.
  - Divided visual standard deviation by $\sqrt{\text{numTags}}$ for multi-tag sights to leverage multi-view geometrical certainties.
