# Roadmap - Milestone v2.5: Hardened EKF Localization & Dynamic Sensor Fusion

## Phase Checklist

- [ ] **Phase 49: State Expansion & IMU Pitch/Roll Telemetry**
  - Add `pitch`, `roll`, linear accelerations, and angular velocities to `DriveState` and actions to enable physical status awareness.
- [ ] **Phase 50: Advanced Outlier Filter (3D Boundaries, Angular Speed, Acceleration/Shock Lockouts)**
  - Implement boundary rejections (out of fields limits, underground Z), gyroscopic speed blur lockouts, and collision-shock lockout logic in `VisionOutlierFilter`.
- [ ] **Phase 51: Dynamic Odometry Covariance Scaling (Tilt/Beaching Protection)**
  - Add protection to `PoseEstimator` that dynamically scales wheel odometry covariance Q under IMU tilt/wheel slippage and freezes odometry if beached.
- [ ] **Phase 52: Dynamic Vision Covariance Scaling (Distance-based SD & Multi-Tag Trust)**
  - Implement dynamic scaling for vision measurement noise R quadratically with distance to the tag, and scale down noise by 1/sqrt(numTags) on multi-tag frame views.
